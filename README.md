# Cash Compass

Финансовый компас — не аналитика, а одно число. Сколько можно потратить сегодня.

Вбиваешь дни зарплаты и текущий баланс — приложение делит баланс на оставшиеся дни. Потратил меньше — завтра число вырастет. Потратил больше — уменьшится. Никаких графиков, категорий и чувства вины.

## Как работает

- Указываешь дни зарплаты (например 12 и 27) в настройках
- Вносишь свободный баланс (за вычетом фиксированных расходов — это ответственность пользователя)
- Видишь одно число — дневной бюджет
- Вносишь траты/приходы через быстрые кнопки или вручную
- Число за сегодня показывается как `факт / план`

## Режимы ввода

- **Изменить** — обычная трата или приход, логируется за сегодня
- **Коррекция** — забытая трата за прошлые дни, пересчитывает якорь без влияния на сегодня
- **Установить** — ручной сброс баланса

## Стек

- React 19 + Vite 6
- PWA (vite-plugin-pwa) — работает оффлайн
- localStorage — никаких бэкендов
- GitHub Pages — https://compass.antoxer.ru

```bash
npm install

npm run build      # продакшн-билд в dist/
npm run preview    # превью продакшн-билда
```
